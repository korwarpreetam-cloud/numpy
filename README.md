# 📊 Sales Data Analysis using NumPy

This is a beginner-level data analysis project where I explored a simple sales dataset using NumPy.

## 🚀 Why I built this

At the beginning, I was confused about how to actually start projects after learning concepts.  
Then I followed a simple roadmap and built this project step by step.

This is one of my first attempts at working with real-like data.

---

## 📂 Dataset

The dataset contains:
- Date
- Product
- Category
- Price
- Quantity

I also created a new column:
- **Revenue = Price × Quantity**

---

## 🛠 What I did

- Loaded CSV data using NumPy
- Cleaned and converted data types
- Calculated:
  - Total revenue
  - Daily sales
  - Product performance
  - Used boolean masking for filtering
- Analyzed sales trends

---

## 📊 Key Insights

- Electronics generated the highest revenue
- Some days had significantly higher sales
- Quantity plays a major role in revenue changes

---

## 🧠 What I learned

- How to work with NumPy arrays in real scenarios
- Data cleaning and type conversion
- Using boolean masking for filtering data
- Performing aggregations like sum and mean
